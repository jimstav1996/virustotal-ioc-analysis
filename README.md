# 🔐 VirusTotal IOC Analysis

## 📌 Overview
This project documents the analysis of a suspicious file using VirusTotal and the Pyramid of Pain model.

## 📧 Scenario
A suspicious password-protected spreadsheet was delivered via email.  
After execution, unauthorized executable files were created on the system.  
An intrusion detection system (IDS) generated an alert.

## 🎯 Objective
- Determine whether the file is malicious  
- Identify indicators of compromise (IoCs)  
- Analyze findings using the Pyramid of Pain  

## 🛠️ Tools Used
- VirusTotal  
- Threat Intelligence Analysis  
- Pyramid of Pain  

## 🧠 Skills Demonstrated
- Malware triage  
- IOC identification  
- Threat analysis  
- SOC investigation workflow
